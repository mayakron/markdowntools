## Features

MarkdownPreviewer and MarkdownConverter are lightweight Markdown tools based on the [Markdig](https://xoofx.github.io/markdig) library.

## Download

All versions of Oracle can be downloaded from [here](https://github.com/mayakron/markdowntools/releases).

## Installation

MarkdownPreviewer and MarkdownConverter are portable: just expand the archive and run the executable files.
